My WoW Addons

Vanilla AddOns

AddOns for Classic World of Warcraft servers like Nostalrius.

NOTE:

  • For an addon to be parsed, the folder name and “addon.toc” must match (e.g. “ColorPickerPlus/ColorPickerPlus.toc”)
  • It’ll be a lot easier to keep up-to-date with these projects if you learn to use Git!

ColorPickerPlus (repo)

Color Picker Plus back-port; original by jaslm (ColorPickerPlus).

ColorPickerPlus hooks into the standard Color Picker to provide

  • text entry for colors (RGB and hex values),
  • text entry for the alpha value (transparency/opacity),
  • copy and paste of a single color,
  • a movable frame,
  • a color swatch for the color that you will be replacing
  • a color swatch for the copied color that may be pasted.

With these enhancements, you can precisely describe a color and coordinate colors within and between add-ons.

A shift-click on the Copy button will paste the decimal RGB values for the chosen color into the chat window. This was added as a utility for add-on developers.

sjChatScan (WIP) (repo)

Provides chat phrase matching and displays matched message as notifications.

DSDS (Dark Souls Death Screen) (repo)

Plays a Dark Souls like “YOU DIED” animation on character death. Accounts for false-flag situations (hunter feign-death, priest spirit of redemption.

This addon was made as an answer to a request in /nosg/ (Nostalrius general on 4chan) and is a complete reimplementation of DarkSoulsDeathScreen by latreese.

sjFont (repo)

Overrides system fonts with custom fonts and scaling. Retains original system font sizes and stylings.

sjMacro (repo)

Provides various functions to simplify mouse-over spell macros, using a priority list to determine the spell target (default: mouse-over > target > player). As a bonus includes and smart-buff feature if playing a class which can buff.

sjUF (Unit Frames) (WIP) (repo)

Simple unit frame overhaul.

sjUI (User Interface) (repo)

Simple user interface overhaul; restyles the following elements:

  • Minimap
  • Action bars and buttons
  • Micro menu buttons
